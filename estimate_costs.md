# My School Hub — Simple Cost and User Capacity Guide

## Purpose of this document

This guide explains:

1. How much hosting is expected to cost each month (in USD).
2. How many user accounts each stage can support smoothly.
3. What happens during busy times like results release.

This version is written for non-technical readers.

## Monthly cost by growth stage (USD)

| Growth stage | Total user accounts | Estimated monthly cost |
|---|---|---|
| Phase 1 | Up to about 2,000 | **$108-$115** |
| Phase 2 | About 2,000 to 5,000 | **~$187** |
| Phase 3 | 5,000 and above | **~$320-$620** |
| Phase 4 | 12,000 to 50,000 | **~$1,150-$2,850** |
| Phase 5 | 51,000 to 100,000 | **~$3,200-$7,900** |

## Why the system can support more users than active users

All registered users are not online at the same time.

In normal school operations:
- Only about **8% to 15%** of total users are active at once.
- Most activity is viewing information.
- A smaller part is updates (for example attendance, marks, payments), mostly done by staff.
- Busy spikes happen around results and exams, but those spikes are short.

Note: WhatsApp is not included in these estimates.

## Smooth capacity by phase

| Phase | Smooth total user accounts | Typical active users at one time | Busy short peak |
|---|---|---|---|
| Phase 1 | **1,500-1,900** | **150-230** | **300-380** |
| Phase 2 | **3,800-4,800** | **380-580** | **750-950** |
| Phase 3 | **9,000-12,000** | **900-1,400** | **1,800-2,400** |
| Phase 4 | **12,000-50,000** | **1,000-7,500** | **up to about 6,000+** |
| Phase 5 | **51,000-100,000** | **4,000-15,000** | **15,000+ supported** |

## What this means for your target ranges

### If you have 12,000 to 50,000 accounts

- You should run on **Phase 4** infrastructure.
- This can comfortably support normal daily school operations and heavy reporting periods.
- Expected cost range: **$1,150-$2,850 per month**.

### If you have 51,000 to 100,000 accounts

- You should run on **Phase 5** infrastructure.
- This is designed for very large, multi-school usage with high peak traffic.
- Expected cost range: **$3,200-$7,900 per month**.

### If you expect over 15,000 users online at the same time

- You must use the **Phase 5** setup with peak-time preparation.
- Capacity is possible, but it requires planned scaling before known busy windows (such as results release).

## Important business notes

1. Costs are estimates and can change with data transfer and usage spikes.
2. The system should be scaled before major growth periods, not after.
3. For best reliability, plan capacity one school term ahead.
